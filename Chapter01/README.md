### 1.2.1 JVMのチューニングフラグ
２種類に分けられる。
- 真偽値フラグ：「-XX:+フラグ名」で有効化、「-XX:-フラグ名」で無効化
- パラメータを必要とするフラグ：「-XX:フラグ名=パラメータの値」で指定
